Callejero vigente de la ciudad: calles o viales, edificios y/o numeraciones, tramos postales y cruces Presenta el listado de viales vigentes (unos 10.000), y portales y/o edificios de la ciudad (unos 200.000), con todo detalle de los mismos: coordenadas geográficas, cruces de viales, tramos de viales con código postal, tramos de vial con zona S.E.R. e incluso población por calles (por seguridad, solamente para aquellas calles que tengan empadronados más de 50 habitantes). License CSV File This dataset is a description of the whole streets, avenues, buildings, etc of the Madrid City. It's part of the smart cities and contains the properties we need to considered to do the practice. There were download the CSV files and have free licenses. The data, could be linked with general real-world entities.
Distritos y barrios (información de perímetro, superficie y geográfica) Códigos y nombres de los distritos y barrios de Madrid, su asociación entre ambos, información de superfície, perimetro e información geografica referida a ED50 y ETRS89. Si se quiere saber la población de un distrito, en el dataset de padrón se puede calcular dicha información. Si se quiere conocer la población de una calle, en el dataset de Callejero Vigente existe un fichero el de viales que indica la población de las mismas (por motivos de seguridad, existe un límite inferior de 50 personas empadronadas en la calle, por debajo de la cual, no se indica la población). Embajadas y Consulados Datos de localización, horarios, contacto y servicios de embajadas y consulados destacados en la ciudad de Madrid. License CSV File Embassys and consulates will be linked with the information of the city. The information provided by the website allow us to use the information. These entities are located in Madrid, and the CSV file contains the street, avenue, number and the geographic coordenates.
Callejero de Zaragoza Servicio que permite a la ciudadanía la localización geográfica de direcciones en un plano de la ciudad de Zaragoza. Ofrece acceso en tiempo real a la información urbana, la integración de información cartográfica de otras administraciones (Catastro, Cartociudad, Sistema de información Territorial de Aragón, Instituto Geográfico Nacional, etc.) y el enlazado a información adicional de las vías de la ciudad a través de Wikipedia. License CSV File It is a description of the plan of Zaragoza city. Thanks to this data, it is possible to know about 3 different sets of information of Zaragoza city: street, portals and semantic maps. It has a free license, and for everyone of these arguments there is a different cvs that we will consult.
Barrios de Zaragoza Conjunto de barrios de Zaragoza License CSV File The districts represent the division of city and the information about this free CVS permits the knowledge of other data in Zaragoza.
Mapas colaborativos Creación de mapas para que la ciudadania pueda añadir información de forma colaborativa. Consulta los mapas que otros ciudadanos han creado y participa en ellos. License CSV File This dataset is interesting because it is possible to know the address and information of the place. The user add new maps and information like the position of museums, schools etc.
Datos Demográficos Totales de Zaragoza Conjunto gráficos, tablas o mapas con datos demográficos sobre la población de la ciudad de Zaragoza, reunidos por el Observatorio de Estadística. License CSV File We will use part of this database for used information about urbanism and infrastructure.
